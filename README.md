# Heedo Creative Studio

AI创意工作室，用于电商广告商。团队上传产品照片，工作室将其转化为广告图片和视频，并保持结果的有序组织。

## 项目概述

### 产品愿景

Heedo正在构建一个AI创意工作室，帮助电商广告团队：

- 上传产品照片
- 生成广告图片和视频
- 保持创作结果的有序组织
- 最终从广告表现中学习

### 核心功能模块

| 模块                 | 描述                                                      | 优先级          |
| -------------------- | --------------------------------------------------------- | --------------- |
| **Canvas (画布)**    | 核心功能 - 节点画布，用户通过连接不同类型节点来组合工作流 | ⭐⭐⭐ 深度实现 |
| **Gallery (画廊)**   | 团队首页，展示最近作品和精选内容                          | 基础实现        |
| **Templates (模板)** | 预设工作流模板，如产品图→场景图→视频的链路                | 基础实现        |
| **Credits (积分)**   | 显示生成成本，管理余额                                    | 基础实现        |
| **Team (团队)**      | 工作空间成员管理和角色权限                                | 基础实现        |

## 技术选型

**前端：** React + TypeScript + Vite
**画布：** React Flow (节点编辑器)
**样式：** Tailwind CSS
**状态管理：** Zustand

> 参考栈：Next.js/TypeScript + FastAPI/Python（非必须）

## 设计决策

### 深度实现部分

1. **Canvas画布** - 完整的节点编辑器
   - 多种节点类型（图片生成、视频生成、滤镜等）
   - 节点连线（类型匹配检查）
   - 实时预览
   - 模拟AI生成

2. **Gallery首页** - 作品画廊
   - 网格展示生成的图片/视频
   - 筛选和搜索功能
   - 快速编辑入口

### 简化实现部分

- Credits系统：显示模拟积分余额
- Templates：提供3-5个预设模板
- Team：基础的成员列表展示

## 开放设计问题 (07)

### 性能数据 × 画布

广告投放后会产生：impressions、CTR、spend、ROAS等数据。

**设计建议：**

1. 在节点上叠加性能指标徽章
2. 侧边栏展示选中创意的详细数据
3. 颜色编码（绿色=好，红色=差）
4. 时间轴对比不同版本表现

**核心原则：** 数据应服务于创意优化，而非喧宾夺主。
