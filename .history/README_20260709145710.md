# Heedo Creative Studio

> AI 创意工作室，专为电商广告团队打造。通过可视化节点画布，将产品照片转化为高质量广告图片和视频。

---

## 项目总体介绍

### 产品定位

Heedo Creative Studio 是一个面向电商广告团队的 AI 驱动创意生产平台。团队成员上传产品照片后，通过可视化工作流编排 AI 能力，生成专业级广告图片和视频素材，并支持团队协作与积分管理。

### 技术架构

```
前端（SPA）
├── Vue 3 + TypeScript        # 框架与语言
├── Vue Flow                   # 节点画布
├── Element Plus               # UI 组件库
├── Pinia                      # 状态管理
└── Vite                       # 构建工具

[后端接口预留]
└── REST API / GraphQL        # 未来对接真实 AI 服务和数据库
```

## 页面功能介绍

### 1. 首页 (Home)

入口页面，展示团队概览数据。

| 功能     | 说明                                               |
| -------- | -------------------------------------------------- |
| 统计卡片 | 总作品数、本月生成、消耗积分、团队成员（模拟数据） |
| 快速入口 | 跳转画布/模板/画廊/积分页面                        |
| 最近作品 | 展示最近 6 个创意作品缩略图                        |
| 创作趋势 | 本周柱状图（静态展示）                             |

### 2. 创意画布 (Canvas)

核心页面，可视化工作流编排器。

#### 2.1 节点库

提供 9 种节点，分为 4 个类别：

| 类别 | 节点        | 积分 | 说明                                                 |
| ---- | ----------- | ---- | ---------------------------------------------------- |
| 输入 | 产品图片    | 0    | 上传本地图片作为工作流起点                           |
| 输入 | 产品描述    | 0    | 输入文本提示词                                       |
| AI   | AI 图片生成 | 50   | 模型（快速/标准/高清）+ 输出尺寸                     |
| AI   | AI 视频生成 | 100  | 时长（2/4/6/10秒）+ 运动模式                         |
| 处理 | 风格滤镜    | 10   | 8 种预设：自然/复古/胶片/黑白/暖调/冷调/HDR/赛博朋克 |
| 处理 | 文字添加    | 5    | 文字内容 + 9 宫格位置选择                            |
| 处理 | 智能裁剪    | 5    | 6 种比例：1:1/4:3/16:9/9:16/3:4/自由                 |
| 输出 | 导出图片    | 0    | 下载按钮 + 预览                                      |
| 输出 | 导出视频    | 0    | 下载按钮 + 视频预览                                  |

#### 2.2 数据类型系统

节点连线需类型匹配：

| 类型   | 标识色       | 说明             |
| ------ | ------------ | ---------------- |
| 图片   | `#4ecdc4` 绿 | 图片类数据传递   |
| 文本   | `#a78bfa` 紫 | 提示词等文本传递 |
| 视频   | `#60a5fa` 蓝 | 视频类数据传递   |
| 无类型 | `#9ca3af` 灰 | 无输出节点       |

#### 2.3 智能连线

- 从输出端口拖拽时，自动弹出兼容节点推荐面板
- 点击即可添加节点并建立连接

#### 2.4 工作流执行

- 点击「开始生成」触发完整执行
- 执行前显示积分消耗预估和余额检查
- 拓扑排序确定节点执行顺序
- 每个节点依次显示 idle → running → done 状态
- 整体进度条实时更新

#### 2.5 草稿管理

- 支持多草稿切换、新建、删除
- 保存草稿至当前会话
- 从模板加载时使用 sessionStorage 中转

#### 2.6 协作与分享

- 生成唯一分享链接（`/canvas/share/:token`）
- 公开访问开关
- 成员角色：owner（拥有者）/ consumer（可消耗点数）/ viewer（仅查看）
- 工作流级可消耗积分额度设置

### 3. 创作模板 (Templates)

预设工作流模板库，支持一键加载到画布。

| 类型   | 模板名称     | 流程                                    | 积分 |
| ------ | ------------ | --------------------------------------- | ---- |
| 短视频 | 基础短视频   | 产品图 → 视频生成                       | 100  |
| 短视频 | 场景化短视频 | 产品图 → 图片生成 → 视频生成            | 150  |
| 短视频 | 风格化短视频 | 产品图 → 图片生成 → 风格滤镜 → 视频生成 | 160  |
| 图片   | 电商主图     | 产品图 → 图片生成 → 智能裁剪            | 55   |
| 图片   | 场景化营销图 | 产品图 → 图片生成 → 风格滤镜            | 60   |
| 图片   | 营销海报     | 产品图 → 图片生成 → 文字添加            | 55   |

**模板特性**：

- 场景提示词在卡片阶段即可编辑
- 积分消耗实时预览
- 流程节点可视化展示
- 一键加载到画布

### 4. 作品画廊 (Gallery)

团队作品展示页面。

| 功能        | 说明                                 |
| ----------- | ------------------------------------ |
| 扇形 Banner | 4 张广告卡片扇形展开动画             |
| 瀑布流      | 不规则网格布局，含随机偏移/缩放/旋转 |
| 搜索过滤    | 实时按标题筛选作品                   |
| 悬停预览    | 缩放 + 渐变遮罩 + 操作按钮           |

### 5. 积分管理 (Credits)

积分余额与消耗管理。

| 功能     | 说明                     |
| -------- | ------------------------ |
| 余额展示 | 渐变数字样式显示当前积分 |
| 使用进度 | 本月消耗进度条           |
| 免费次数 | 每日免费生成次数         |
| 消耗明细 | 表格展示积分流水记录     |
| 充值套餐 | 4 档套餐，含赠送积分     |

### 6. 团队管理 (Team)

团队成员管理。

| 功能     | 说明                             |
| -------- | -------------------------------- |
| 成员列表 | 头像、名称、邮箱、在线状态       |
| 角色管理 | admin / editor / viewer 三级权限 |
| 权限说明 | 各角色能力可视化展示             |
| 邀请入口 | 模拟添加新成员                   |

### 7. 权限管理 (Permissions)

工作流级权限精细化控制。

| 功能     | 说明                               |
| -------- | ---------------------------------- |
| 权限说明 | owner / consumer / viewer 能力对比 |
| 额度控制 | 工作流可消耗积分上限设置           |
| 成员列表 | 角色切换、移除操作                 |
| 邀请入口 | 邮箱邀请 + 角色选择                |

## 模拟数据规范

> 以下所有标注 `[MOCK]` 的模块为前端模拟实现，未来可无缝替换为真实后端服务。

### 模拟模块清单

| 模块         | 文件位置             | 模拟内容        | 未来 API 格式                                                |
| ------------ | -------------------- | --------------- | ------------------------------------------------------------ |
| **用户认证** | `App.vue`            | 随机生成 userId | `POST /auth/login` → `{ userId, token, role }`               |
| **积分数据** | `stores/creative.ts` | Pinia ref       | `GET /credits` → `{ balance }`                               |
| **积分消耗** | `CanvasView.vue`     | 预扣费确认      | `POST /credits/deduct` → `{ success, balance }`              |
| **充值功能** | `CreditsView.vue`    | 直接加积分      | `POST /credits/recharge` → `{ balance, transactionId }`      |
| **交易记录** | `stores/creative.ts` | 预置 4 条记录   | `GET /credits/transactions` → `[{ id, desc, amount, time }]` |
| **作品数据** | `stores/creative.ts` | 8 条预置 + 本次 | `GET /creatives` → `[{ id, title, type, status }]`           |
| **模板加载** | `TemplatesView.vue`  | sessionStorage  | `GET /templates` → `[{ id, name, nodes, edges }]`            |
| **团队成员** | `TeamView.vue`       | 4 条预置数据    | `GET /team/members` → `[{ id, name, email, role }]`          |
| **在线状态** | `TeamView.vue`       | 随机布尔值      | `WS /team/presence` → `{ userId, status }`                   |
| **分享链接** | `CanvasView.vue`     | 随机字符串      | `POST /workflows/:id/share` → `{ token, url }`               |
| **AI 生成**  | `CanvasView.vue`     | picsum 模拟     | `POST /ai/generate` → `{ imageUrl, videoUrl, cost }`         |
| **首页统计** | `HomeView.vue`       | 预置数字        | `GET /dashboard/stats` → `{ totalWorks, thisMonth, spent }`  |
